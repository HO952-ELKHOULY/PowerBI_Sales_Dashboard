# Sales Analysis Dashboard - Power BI

# Overview
This Power BI dashboard provides an interactive view of sales performance across countries, years, customers, and deal sizes. It is designed for business insights with a focus on clear KPIs, user-friendly filtering, and easy reset functionality.

# Key Features
- Dynamic KPIs: Key performance indicators automatically update based on filter selections.
- Global Reset Button: Quickly reset all filters and slicers using bookmarks.
- Sync Filtering Across Pages: Filter selections apply consistently across all report pages.
- Conditional Formatting: Highlights key metrics such as top customers and deal sizes.

# How to Use
1. Open the `.pbix` file in Power BI Desktop.
2. Use the slicers to filter by Country, Year, or Customer.
3. Press the **Reset** button to return all filters to default.
4. Navigate between pages and observe synced filtering and dynamic KPIs.

# Requirements
- Power BI Desktop (latest version)
- Windows 10 or later (recommended)
